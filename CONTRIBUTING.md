# 贡献指南（简版）

欢迎提交新增和修订。

## 最简单流程

1. Fork 仓库
2. 修改 `nav.html`
3. 提交 PR

## 只保留 4 条规则

1. 不做无关的大规模格式化
2. 新增链接尽量放到正确分类
3. 避免重复链接
4. PR 描述里写清楚你改了什么

## 新增链接建议格式

```html
<a href="https://example.com/" target="_blank" rel="noopener" class="bk-card" title="示例站点">
  <span class="bk-name">示例站点</span>
</a>
```
